# Abrechnungen importieren

## Unterstützte Formate

Die App unterstützt zwei Importformate:

| Format | Beschreibung |
|--------|--------------|
| **CSV** | Transaktionsliste von der Bank |
| **PDF** | Kreditkartenabrechnung als PDF |

## CSV-Datei von der Bank herunterladen

### American Express
1. Melde dich im Online-Banking an
2. Gehe zu "Kontoauszüge" → "Transaktionen herunterladen"
3. Wähle den Zeitraum und Format "CSV"
4. Speichere die Datei

### Visa / Mastercard (DKB)
1. Melde dich im DKB Online-Banking an
2. Wähle die Kreditkarte aus
3. Klicke auf "Umsätze exportieren" → "CSV"
4. Speichere die Datei

## Import durchführen

1. Klicke auf den **Import**-Button
2. Wähle die passende **Kreditkarte** aus der Liste
3. Die **Periode** wird automatisch vorgeschlagen (z.B. "Januar 2026")
   - Du kannst sie anpassen falls nötig
4. Ziehe die Datei in den Upload-Bereich oder klicke zum Auswählen
5. Klicke auf **Importieren**

## Nach dem Import

Nach erfolgreichem Import siehst du:
- Anzahl der importierten Transaktionen
- Gesamtbetrag der Abrechnung
- Eventuelle Gutschriften

Die neue Abrechnung wird automatisch ausgewählt und die Transaktionen werden angezeigt.

## Tipps

- **Doppelte Importe** werden erkannt – bereits vorhandene Transaktionen werden übersprungen
- **PDF-Import** extrahiert die Transaktionen automatisch per OCR
- Bei Problemen prüfe, ob das richtige Konto ausgewählt ist
