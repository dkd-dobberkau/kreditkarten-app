# Kategorisierung

## Automatische Kategorisierung

Beim Import werden Transaktionen automatisch kategorisiert. Die KI analysiert:
- Händlername
- Betrag
- Bekannte Muster aus früheren Buchungen

### Verfügbare Kategorien
- Reisekosten
- Bewirtung
- Büromaterial
- Software & Lizenzen
- Hardware
- Telekommunikation
- Sonstiges

## Kategorie manuell ändern

1. Klicke auf die Transaktion
2. Wähle im Dropdown **Kategorie** die richtige aus
3. Die Änderung wird sofort gespeichert

## Kategorie-Regeln

Für wiederkehrende Händler kannst du Regeln erstellen:

1. Klicke auf eine Transaktion
2. Wähle die richtige Kategorie
3. Aktiviere **"Regel erstellen"**
4. Ab sofort werden alle Buchungen von diesem Händler automatisch dieser Kategorie zugeordnet

### Regeln verwalten
- Gehe zu **Einstellungen** → **Kategorie-Regeln**
- Hier siehst du alle aktiven Regeln
- Du kannst Regeln bearbeiten oder löschen

## Beispiele für Regeln

| Händler enthält | Kategorie |
|-----------------|-----------|
| DEUTSCHE BAHN | Reisekosten |
| AMAZON WEB SERVICES | Software & Lizenzen |
| OFFICE DEPOT | Büromaterial |

## Tipps

- Die KI lernt aus deinen manuellen Korrekturen
- **Konsistente Kategorisierung** hilft bei der Auswertung
- Bei Unsicherheit: "Sonstiges" wählen und später korrigieren
